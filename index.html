Technologies

Python

Python is a high-level, interpreted programming language known for its simplicity and readability. Created by Guido van Rossum and released in 1991. it emphasizes clean syntax and code readability, making it relatively easy to learn and use, even for beginners

Javascript

JavaScript is a high-level, interpreted programming language primarily known for enabling interactive and dynamic content on websites. It forms the third layer of standard web technologies, working alongside HTML (which provides the structure and content) and CSS (which handles the styling and layout).

ReactJS

React.js (also known as React or ReactJS) is an open-source JavaScript library for building user interfaces (UIs), particularly single-page applications. It was created by Facebook and is now maintained by Meta and a community of developers
